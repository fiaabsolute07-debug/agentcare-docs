# Publishing a skill

Skills are reusable building blocks you can share with the community. This guide explains the types and the publishing flow. You need to be logged in to publish.

## What a skill is

A skill is something other builders and agents can reuse. There are three types.

A Prompt is a reusable prompt template for agents.

A Tool is a function or API endpoint that agents can call.

A Workflow is a multi-step chain that combines prompts and tools.

## The publish flow

In the app, open Publish Skill. The flow has three steps.

### Step 1 - Type

Choose whether you are publishing a prompt, a tool, or a workflow.

### Step 2 - Content

Fill in the details of your skill.

- Title. The name of your skill.
- Description. What it does and when to use it.
- Content or code. The actual prompt text, the tool definition, or the workflow steps.
- Tags. Comma separated keywords to help others find it.

A live preview shows how your skill will appear as you fill in the form.

### Step 3 - Publish

Review the summary, then publish. Your skill becomes available to the community. You can also save it as a draft if you are not ready.

## Beta limits

During beta, accounts can publish up to a set number of skills. Check your usage in your account under Plan and Usage.
