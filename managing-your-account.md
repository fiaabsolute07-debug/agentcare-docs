# Managing your account

Your account is where you control your profile, plan, API keys, and notifications. This guide covers each area. You need to be logged in.

## Profile

Set how you appear in Agent Care. You can change your face, email, handle, display name, and bio. Handles are 3 to 20 characters, lowercase, using letters, numbers, and underscores.

## Plan and usage

See your current plan and how much of it you are using.

During beta, the plan includes a set number of agents, skills, and API keys, plus community support. Your usage shows how many agents, skills, and keys you have created against those limits.

## API keys

Create and manage the keys used to authenticate with Agent Care programmatically. If you have no keys yet, you can create one here. Treat API keys like passwords - do not share them or commit them to a public repository.

For how keys are used in requests, see the [API reference](../api-reference/overview.md). Note that authentication details are subject to change during beta.

## Notifications

Choose what you want to be told about, including agent status updates, community replies, demo request alerts, billing and usage warnings, and product updates.

## Danger zone

This area holds permanent actions. You can export your data, or delete your account. Deleting your account is permanent and removes all your agents, data, and keys. Be sure before you proceed.
